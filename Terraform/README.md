# Terraform

1. Initalize folder wihere `.tf` files will be:

```sh
terraform init
```

2. Show plan of Terraform:

```sh
terraform plan
```

3. If everything Ok apply terraform plan:

```sh
terraform apply
```

Write `yes` and check results.

4. When your done you can destroy infrastructure:

```sh
terraform destroy
```

5. Useful commands

```sh
terraform list
```
```sh
terraform show
```
